# Node Web Scraper

- scrape a page using request, cheerio.
- using es6 and [babel](https://gist.github.com/vanessachem/d071027fad39507b9c9f)
- using one page from imdb as a example
- output to json file in current repo

## start up 
```javascript	
npm start
```
